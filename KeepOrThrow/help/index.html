Spielprinzip:


Auf dem gemischten Stapel befinden sich eine zufällig erzeugte Anzahl von Karten (52-12).

Die oberste Karte kann entweder behalten(keep) oder weggeworfen(throw) werden.

Auf eine behaltene Karte darf entweder eine Karte mit höherem Wert oder

mit gleichem Wert und Typ gelegt werden.

Ziel des Spiels

Das Deck soll in möglichst kleiner Zeit möglichst groß werden.

Ende des Spiels:

Das Spiel ist beendet, wenn das gemischte Deck leert ist.
Wenn die Zeit um ist, wird das gemischte Deck auatomatisch geleert.
 

Tastaturkürzel:
